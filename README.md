# Multiple Linear Regression in Machine Learning

## 📌 Project Overview

This project demonstrates the implementation of **Multiple Linear Regression (MLR)**, a supervised machine learning algorithm used to predict a target variable using multiple independent features.

Unlike Simple Linear Regression, which uses a single predictor, Multiple Linear Regression analyzes the combined effect of several features on the target variable, helping build more accurate and realistic predictive models.

---

## 🎯 Objectives

* Understand the concept of Multiple Linear Regression
* Learn how multiple features influence a target variable
* Build and train a regression model
* Evaluate model performance using regression metrics
* Interpret model coefficients and predictions

## ⚙️ Implementation Steps

### Data Preparation

* Load and inspect the dataset
* Handle missing values if required
* Separate features and target variable

### Model Building

* Train the Multiple Linear Regression model
* Learn regression coefficients

### Prediction

* Generate predictions on test data
* Compare actual and predicted values

### Model Evaluation

* Calculate Mean Absolute Error (MAE)
* Calculate Mean Squared Error (MSE)
* Calculate R² Score
* Analyze model performance
